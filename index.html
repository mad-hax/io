Welcome

We live to develop both open source and properietary software in order to:

  • Help establish structure in a world of chaos and entropy
  • Architect and quantify highly complex, multi-variant systems which model or simulate reality
  • Create truly world-positive tools
  • Unite people by promoting free speech and self-hosted services
  • Decentralize content and distribute trust
  • Facilitate advanced communication and coordination between heterogenous actors
  • Ease and assist the development of other world-positive tools

Services

If your goals are world-positive, there are many ways in which we can help you achieve them:

  • Open-source project contributions
  • Platform architecture
  • RESTful API design
  • UI and UX development
  • Automated testing
  • Documentation
  • Code reviews
